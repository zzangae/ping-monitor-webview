### 프로그램 시작

1. **ping_monitor.exe 실행**
   - 더블클릭 또는 명령줄에서 실행
   - 트레이 아이콘 생성
   - 브라우저 자동 오픈 (http://localhost:8080)

2. **트레이 아이콘 메뉴**
   - 좌클릭 더블클릭: 브라우저 열기
   - 우클릭: 컨텍스트 메뉴
     - 시작/일시정지
     - 브라우저 열기
     - 알림 활성화 (토글)
     - 포트 변경
     - 종료

### 대시보드 사용

#### 📊 대시보드 탭

- **통제 바**
  - 현재 시간/날짜 표시
  - 모니터링 경과 시간
  - 일시정지/재개 버튼
  - 새로고침 버튼

- **IP 카드**
  - 상태 표시: 온라인(🟢) / 오프라인(🔴)
  - 지연 시간 (ms)
  - 통계: 전체/성공/최소/최대/평균
  - 실시간 그래프 (60개 데이터 포인트)
  - 최소화 버튼 (─)

- **필터 버튼**
  - 전체: 모든 IP 표시
  - 내부: 내부 IP만 (192.168.x.x, 10.x.x.x 등)
  - 외부: 외부 IP만
  - 오프라인: 응답 없는 IP만

- **드래그 앤 드롭**
  - IP 카드를 드래그하여 순서 변경
  - localStorage에 자동 저장

#### 🔔 알림 기록 탭

- 타임아웃/복구 알림 이력 표시
- 시간, IP, 이름, 타입별 정렬
- 날짜별 필터링

## 💡 사용 팁 : v2.6

### 장애 관리
- 중요한 서버는 임계값을 짧게 (2-3분)
- 일반 장비는 기본값 (5분) 사용
- `outage_log.json`을 정기적으로 분석

### 설정 UI
- 대시보드에서 직접 수정 가능
- 변경 후 "Save" 버튼 (향후 구현 예정)
- 현재는 `ping_config.ini` 직접 수정

### 설정 불러오기
- IP 추가/삭제 후 바로 적용
- 프로그램 재시작 불필요
- 브라우저 새로고침 (F5) 필요

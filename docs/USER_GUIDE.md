## 5. USER_GUIDE.md

**포함 내용:**

- 프로그램 시작/종료
- 트레이 아이콘 사용
- 대시보드 기능
- IP 카드 조작
- 필터링 사용
- 알림 설정
- 포트 변경

### 프로그램 시작

1. **ping_monitor.exe 실행**
   - 더블클릭 또는 명령줄에서 실행
   - 트레이 아이콘 생성
   - 브라우저 자동 오픈 (http://localhost:8080)

2. **트레이 아이콘 메뉴**
   - 좌클릭 더블클릭: 브라우저 열기
   - 우클릭: 컨텍스트 메뉴
     - 시작/일시정지
     - 브라우저 열기
     - 알림 활성화 (토글)
     - 포트 변경
     - 종료

### 대시보드 사용

#### 📊 대시보드 탭

- **통제 바**
  - 현재 시간/날짜 표시
  - 모니터링 경과 시간
  - 일시정지/재개 버튼
  - 새로고침 버튼

- **IP 카드**
  - 상태 표시: 온라인(🟢) / 오프라인(🔴)
  - 지연 시간 (ms)
  - 통계: 전체/성공/최소/최대/평균
  - 실시간 그래프 (60개 데이터 포인트)
  - 최소화 버튼 (─)

- **필터 버튼**
  - 전체: 모든 IP 표시
  - 내부: 내부 IP만 (192.168.x.x, 10.x.x.x 등)
  - 외부: 외부 IP만
  - 오프라인: 응답 없는 IP만

- **드래그 앤 드롭**
  - IP 카드를 드래그하여 순서 변경
  - localStorage에 자동 저장

#### 🔔 알림 기록 탭

- 타임아웃/복구 알림 이력 표시
- 시간, IP, 이름, 타입별 정렬
- 날짜별 필터링
